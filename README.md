# EinkTest_APP

#### 介绍

此程序绝大部分由DeepSeek生成

功能：
1.VCOM连接测试（当时写这个程序的初衷）
元太的新屏幕，特别是彩色滤光片屏，常见屏幕表层VCOM层到与排线上的阻值增高，甚至断开，导致VCOM不能保持，刷屏异常
最常见于EC078KH4,EC078KH6这两款屏幕，7寸carta1200的黑白和彩色也有见过
此功能可以在不严重时就检测出问题
2.屏幕帧率检测
有普通方法，就是方块或者圆循环显示的方法，在普通显示器上或许这是比较好的方法了，不过墨水屏刷新率低，实在不是很能视觉暂留
所以做了墨水屏的专用方法，就是利用残影，1秒钟刷一遍，看看留下来多少个残影就是了，这里做了黑白屏和彩色屏的专门优化方法，彩色屏因为滤光片分布从颜色刷到黑色残影是很大的，黑白屏比较麻烦，最后只能是用竖线格做底色还是可以看到残影的
3.屏幕坏点/安装位置检测
就是白屏，黑屏，带一圈线
4.灰度测试
横着的灰度分布和竖着的灰度分布来回切换，测试灰度刷新效果用的
5.方格刷新测试
烤屏功能，别开太久，做供电稳定性测试的
点击界面显示出下面调节方格大小和帧率的按钮
